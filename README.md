# **FormalAutomaton**

Automat inputs in the following way:

-Sigma(starts from 1)

-Number of edges

-All edges in format (from, char, to)

-Start vertex

-Number of terminal vertices

-All terminal vertices


-----------------------------------------------

## Launching:

mkdir build

cd build

cmake ../ && make && ./MainExe


-----------------------------------------------

## Lauching tests:

mkdir build

cd build

cmake ../ && make && ./CTestsExe
